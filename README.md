# LivasModLoader

LivasModLoader — современный Windows-лаунчер, менеджер модов и менеджер сервера для **Chivalry 2** с космической темой интерфейса. Приложение построено на **.NET 8 + WPF + MVVM** и ориентировано на безопасное управление файлами (.pak) и запуск игры через официальные методы.

## Возможности

- Лаунчер с тремя профилями запуска (обычный, с модами, Unchained).
- Управление модами: библиотека .pak, включение/отключение, обновления по URL.
- Менеджер выделенного сервера с предпросмотром аргументов запуска.
- Настройки платформы (Steam/Epic), путей и обновлений.
- Прозрачные уведомления о действиях приложения.

## Сборка и запуск

Требования:
- Windows 10/11
- .NET 8 SDK

Команды:

```bash
cd LivasModLoader

dotnet restore

dotnet build

dotnet run
```

## Конфигурация

Файлы конфигурации и состояния сохраняются в `%AppData%\LivasModLoader`:

- `config.json` — пути, настройки платформ и параметры запуска.
- `state.json` — список файлов, синхронизированных в игру, и отметки запусков.

## Безопасность

- Не используется инжекция DLL, не выполняются скрытые действия.
- Менеджер модов работает только с файлами `.pak` и метаданными `.json`.
- Перед запуском игры и сервера показываются явные пути и настройки.
